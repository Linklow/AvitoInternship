# AvitoInternship

Weather app.

Libraries used: Retrofit, Coroutines, RecyclerView, LiveData, Gms Location.

An application to find out the weather for the day and for the week.
Click "See your city" to see the weather in your city.
Write in the input field the name of the city (for example "Moscow") to see the weather in this city.
